# Poke-Project
Juan Sebastián Herrera Suarez, Johan Stiveen Dueñas Lopez, Jose David Tarazona Diaz

![Poke](poke002.jpg)  
**ODJETIVO:** La predicción de pokemon's legendarios, en base a sus estadisticas.

- **DataSet:** Contiene los datos de 1025 Pokemon's diferentes, con todos los datos pertinentes: Tipo, Estadisticas, Generacion de aparición, entre otros.
- **Librerias:** sklearn, tensorflow, pandas, numpy, seaborn, matplotlib.
